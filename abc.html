The keyboard and mouse are currently the main interfaces between man and computer. With the advancement in technology the traditional hardware keypads were replaced by touch screens. The basic idea behind this is to ease the interaction between a user and a computer. With our project we tried to implement an easy and cheap interaction interface ( using movement of color objects in air ) between the digital information and the physical world. The concept of intelligence in the project is implemented using Color Recognition and Detection for mouse control and Optical Character Recognition for Keyboard Implementation.All we need is a computer, camera, colored objects and MatLab.

Detection and Recognition are the two independent processes in a color recognition system working in real time. This is achieved using system dependent and application oriented tasks such as image capturing, extraction of desired color components and classification of the information. On detecting and recognizing a particular color (Red -> "track" , Blue -> "click" and Green -> "scroll") and number of colored objects ( 1 blue color - perform left click, 2 blue colors - perform right click, 3 blue colors - perform double click ) the corresponding mouse functions ( mouseMove and mousePress ) are initiated using Java Robot class

Recognition and identification of alphabets (A, B, C, D, E, F) is implemented using the Neural Network Toolbox in MatLab. The alphabet written in air is acquirred by the system, segmented using the image processing toolbox and fed to the nueral network toolbox. We have used the Feed Forward Back Propagation model.
